# Library Management System

A database-driven application developed using Java and MySQL to manage books, members, and borrowing activities in a library.

## Tech Stack
- Java
- JDBC
- MySQL
- SQL

## Features
- Add and manage books
- Register library members
- Issue and return books
- Fine calculation for late returns
- Database-driven storage using MySQL

## Modules
1. Book Management
2. Member Management
3. Issue/Return System
4. Fine Calculation

## Tools Used
- MySQL Workbench
- JDBC
- Git & GitHub

## Project Status
Currently under development. Additional features and improvements will be added.
